# Paradigm Pet Professionals - Website Redesign

**WGU UX/UI Project**

This repository contains the completed Task 1 and Task 2 deliverables for the Paradigm Pet Professionals website redesign project. The goal of this project was to evaluate the company's existing website, define improved information architecture, and produce a fully interactive desktop prototype aligned with audience personas and stakeholder requirements.

This repo also serves as a polished case study demonstrating UX research, IA development, wireframing, and interface design.

##  Project Overview

**Paradigm Pet Professionals** is a company specializing in providing virtual consultations and evidence-based pet care guidance. Their existing website was outdated, text-heavy, and did not meet modern usability standards.

### This project involved:

- Evaluating the existing site
- Identifying accessibility and navigation issues
- Analyzing audience needs and stakeholder goals
- Creating a new information architecture
- Designing mid-fidelity wireframes
- Building a functional interactive prototype

---

##  Task 1 Summary - Research + Architecture

Task 1 involved analyzing the current site and producing UX documentation, including:

### ✔ Evaluation of Current Website
Assessed content, functionality, and navigation against user and stakeholder expectations.

### ✔ Audience & Stakeholder Needs
Defined based on provided personas and business objectives.

### ✔ Information Architecture
Redesigned site structure, including:
- Home
- Dog Owners
- Cat Owners
- New Persona Page (Bird Owners)
- Consultation Form

### ✔ Visual Sitemap
A comprehensive sitemap that showcases an enhanced hierarchy and navigation paths.

### ✔ Mid-Fidelity Wireframe (Home Page)
Desktop wireframe designed to reflect improved layout and functionality.

### ✔ Maintenance Plan
Outlined ongoing tasks for accessibility, SEO, updates, and responsive behavior.

---

##  Task 2 Summary - Interactive Prototype

Task 2 involved creating a multi-page, interactive desktop prototype that includes:

### ✔ Updated Pages
- Home Page
- Dog Owners Page
- Cat Owners Page

### ✔ New Persona Page
- Bird Owners Page

### ✔ Consultation Request Form
Includes all required fields:
- Name
- Email
- Phone
- Time Zone
- Pet Name
- Pet Type
- Pet Age

### ✔ Navigation + Positional Awareness
- Consistent global navigation
- Active link states for "current page" awareness
- Fully clickable paths between all sections

### ✔ Consistent Design System
Typography, spacing, card layout, buttons, and imagery adhere to a consistent visual language for clarity and usability.
